# 80-claude-code

Instala o Claude Code CLI via installer oficial: `curl -fsSL https://claude.ai/install.sh | bash`.

Cross-OS — mesmo script para WSL e Mac. O binário é instalado em `~/.local/bin/claude` (que o `30-shell` já coloca no `PATH`).

**Login:** após instalar, rodar `claude` pela primeira vez para autenticar.
